# Cursor Official Themes

Official [Cursor](https://www.cursor.com) themes ported to VS Code with small modifications to fit the VS Code environment.

## Themes

| Theme | Type |
|-------|------|
| **Cursor Dark** | Dark |
| **Cursor Dark Midnight** | Dark |
| **Cursor Dark High Contrast** | High Contrast |
| **Cursor Light** | Light |

## Installation

1. Copy this extension folder to your VS Code extensions directory, and make sure the folder name starts with `local.` (for example: `local.cursor-official-themes-1.0.0`):
   - **macOS/Linux:** `~/.vscode/extensions/`
   - **Windows:** `%USERPROFILE%\.vscode\extensions\`
2. Restart VS Code or reload the window (`Cmd+Shift+P` / `Ctrl+Shift+P` -> **Developer: Reload Window**).

## Activating a Theme

1. Open the Command Palette (`Cmd+Shift+P` / `Ctrl+Shift+P`)
2. Run **Preferences: Color Theme**
3. Select one of the **Cursor** themes from the list
